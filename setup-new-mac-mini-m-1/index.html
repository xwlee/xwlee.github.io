Setup New Mac Mini M1

28.03.2021 — macmini — 1 min read

Recently, I just brought a new Mac Mini with the M1 chip. Thought it would be nice to make a post about how I set up my development machine from the ground up.

For the software compatibility, you can use this website to check.

Install Homebrew

The first software to install for me always is Homebrew.

1$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install git & vim & openjdk

1brew install git
2brew install vim
3brew install openjdk

Setup Git config

1git config --global user.name "Lee Xiang Wei"
2git config --global user.email "xwlee2009@gmail.com"
3# Aliases
4git config --global alias.co checkout
5git config --global alias.br branch
6git config --global alias.ci commit
7git config --global alias.st status
8git config --global alias.unstage 'reset HEAD --'

Install Node.js via nvm

1brew install nvm
2# You should create NVM's working directory if it doesn't exist:
3mkdir ~/.nvm

Add the following to ~/.zshrc or your desired shell configuration file:

1export NVM_DIR="$HOME/.nvm"
2[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"  # This loads nvm
3[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"  # This loads nvm bash_completion

Install LTS version node.

1nvm install --LTS

Install yarn.

Install prezto

prezto is the configuration framework for Zsh shell.

It comes with modules and themes that allow you to customize your terminal.

To install prezto, run

1git clone --recursive https://github.com/sorin-ionescu/prezto.git "${ZDOTDIR:-$HOME}/.zprezto"

Copy the configuration files

1setopt EXTENDED_GLOB
2for rcfile in "${ZDOTDIR:-$HOME}"/.zprezto/runcoms/^README.md(.N); do
3  ln -s "$rcfile" "${ZDOTDIR:-$HOME}/.${rcfile:t}"
4done

Append this to your .zprofile to make sure brew binary can be called when open new terminal.

1eval $(/opt/homebrew/bin/brew shellenv)

Update .zprofile and change below section to vim

1...
2export EDITOR='vim'
3export VISUAL='vim'
4...

Select a theme for your terminal

To list all the available themes

1prompt -l

To preview a theme

1prompt -p <theme>

To set a theme

1prompt -s <theme>

Install a module for your terminal

You can browse here for the available modules

To add a new module, open ~/.zpreztorc and add the module to the following section

1...
2zstyle ':prezto:load' pmodule \
3  'environment' \
4  'terminal' \
5  'editor' \
6  'history' \
7  'directory' \
8  'spectrum' \
9  'utility' \
10  'completion' \
11  'prompt' \
12  # Add new module here
13...

Then you need to reload ~/.zpreztorc

1source ~/.zpreztorc
